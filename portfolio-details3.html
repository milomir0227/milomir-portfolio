Project information

  • Category: Social App
  • Stacks: React, Nextjs, Laravel, MySQL, AWS, Git
  • Project URL: nexthuman.me

Laravel: We used Laravel, a PHP-based web application framework, to develop the backend of the website. Laravel provides a robust set of tools for building web applications quickly and efficiently, and it allows for easy integration with other technologies.

React.js: We used React.js, a popular front-end JavaScript library, to develop the user interface and user experience of the website. React.js allows for the creation of dynamic, interactive, and responsive user interfaces that provide a seamless experience for our customers.

MySQL: We used MySQL as the database management system to store and retrieve website data, such as user information, golf course listings, and tee time bookings.

AWS: We used Amazon Web Services (AWS) to host our website and ensure scalability, reliability, and security for our web hosting infrastructure.

Git: We used Git for version control to manage and collaborate on the project's source code with other developers.
