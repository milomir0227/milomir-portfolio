Project information

  • Category: Social App
  • Stacks: Nuxtjs, Vue, Laravel, AWS, MySQL
  • Project URL: maven.aju.edu

Laravel: We used Laravel, a PHP web application framework, to develop the backend of our website. Laravel allowed us to easily handle user authentication, data validation, and database management.

Vue.js: We used Vue.js, a progressive front-end JavaScript framework, to build the user interface and user experience of the website. Vue.js allowed us to build dynamic and reactive user interfaces that are both engaging and efficient for our users.

MySQL: We used MySQL as the database management system to store and retrieve website data such as user information, service provider information, and booking data. MySQL is a reliable and widely-used database management system that allowed us to manage the large amounts of data generated by our platform.

AWS: We used Amazon Web Services (AWS) to host our website and ensure scalability, reliability, and security for our web hosting infrastructure.

Git: We used Git for version control to manage and collaborate on the project's source code with other developers.
